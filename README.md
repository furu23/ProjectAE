
# ProjectAE

Developed with Unreal Engine 5.4.4

### 프로젝트 개요
- **프로젝트 이름**: Alien Escape
- **게임 장르**: TPS 익스트렉션 슈터
- **프로젝트 기간**: 2025.11.03 - 2025.12.24 (약 50일)
- **플랫폼**: Windows

### TeamAE
- 김재환: PM, 인게임 게임플레이 개발
- 김영채: 적 AI 개발, 레벨 디자인
- 박동진: 코어 게임시스템 개발

### 게임 조작법
- 이동: `W A S D`
- 달리기: `SHIFT`
- 구르기: `SPACE`
- 상호작용: `F`
- 인벤토리: `I`
- 퀘스트: `P`
- 일시정지: `ESC`
